###. 노란색 사각형(유휴상태)과 회색사각형(실행상태)에 주의를 기울여야 한다. 이 모두는 방금 만든 에니메이션이다. 유휴 애니메이션은 애니메이터의 기본 상태로 설정 되어 있기 때문에 주황색이다.

###. 현재 유휴 상태를 오른쪽 버튼을 클릭하고 전환만들기[Make Transition]를 누른다.
